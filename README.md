# Lenga-Maps
Public overview and documentation for Lenga Map's  geospatial data platform, starting with administrative boundaries.
## Phases of the geodatabase will be built in this order:
1. Administrative Boundaries
2. Water Data (Lakes, Rivers and watersheds)
3. Soils
4. LandUse (can be updated every 1 year 6 months)
5. DEMs (Digital Elevation Models)
6. and more...
